O que é animação?

Animação é uma técnica que cria a ilusão de movimento através de uma sequência de imagens. Cada quadro possui pequenas mudanças que, juntas, formam uma cena animada.

Tipos de Animação

Animação 2D

Criada através de desenhos em duas dimensões, tradicional ou digital.

Animação 3D

Utiliza modelos digitais para criar profundidade e realismo.

Stop Motion

Usa fotos em sequência de objetos físicos para criar movimento.

Como começar uma animação?

  1. Criar uma ideia ou conceito
  2. Escrever uma história (roteiro)
  3. Fazer um storyboard
  4. Criar as animações quadro a quadro
  5. Editar o resultado final (som, efeitos, etc)
